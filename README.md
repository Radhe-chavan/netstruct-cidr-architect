# NetStruct - CIDR Architect

NetStruct is a precision-engineered network topology and CIDR analysis platform. It provides mission-critical tools for subnet planning, broadcast domain management, and cloud VPC optimization.

## 🛡️ Core Mandates
- **Bitwise Precision**: Local-first JavaScript engine for accurate RFC-compliant subnet calculations.
- **Privacy Centric**: All infrastructure planning data remains local to your browser; no enterprise IP entries are transmitted or stored.
- **CLS Prevention**: Strict structural ID hooks (`#slot-lead-billboard`, `#slot-sidebar-skyscraper`) enforce layout stability for AdSense integration.
- **Semantic SEO**: Standard HTML5 hierarchy for maximum crawlability of our technical Knowledge Base.

## 🚀 Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Verify `ads.txt` in the `/public` directory.

## 📂 Architecture
- `/src/app/articles`: Technical Knowledge Base index and articles.
- `/src/components/cidr-calculator.tsx`: Core bitwise calculation engine.
- `/src/app/legal`: Sanitized legal pages (Privacy, Disclaimer, Rights) tailored for network infrastructure.
